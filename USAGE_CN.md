# 使用指南

## 快速开始

### 第一步：安装扩展

1. 下载或克隆本项目到本地
2. 打开 Chrome 浏览器
3. 地址栏输入 `chrome://extensions/` 并回车
4. 打开右上角的"开发者模式"
5. 点击"加载已解压的扩展程序"
6. 选择本项目文件夹

### 第二步：配置 API Key

1. 点击扩展图标，在弹出窗口点击"⚙️ 设置"
2. 前往 [DeepSeek 平台](https://platform.deepseek.com) 注册账号
3. 在控制台创建 API Key
4. 将 API Key 复制粘贴到设置页面
5. 点击"保存设置"

### 第三步：提取评论

1. 打开小红书或抖音的任意页面（需要有评论区）
2. 点击浏览器工具栏的扩展图标
3. 点击"提取评论"按钮
4. 等待提取完成，查看评论列表

### 第四步：AI 分析（可选）

1. 在成功提取评论后，点击"🤖 AI 分析"按钮
2. 等待 AI 分析完成（通常需要几秒钟）
3. 查看分析结果，显示识别出的汽车相关用户

## 功能详解

### 评论提取

- **支持平台**: 小红书、抖音
- **提取内容**: 用户名、评论内容、发布时间
- **显示方式**: 表格形式，清晰易读

### AI 智能分析

AI 会自动识别以下汽车相关内容：

1. **明确的汽车内容**
   - 关键词：汽车、车辆、车型、SUV、轿车等
   - 讨论：驾驶、保养、改装、维修等

2. **购买意向**
   - 价格讨论："这个多少钱"、"6800我要了"
   - 交易意向："出吗"、"5万收"、"能便宜吗"

3. **品牌咨询**
   - 品牌询问："有比亚迪吗"、"有长安没"
   - 车型咨询："这是什么型号"、"能上牌吗"

4. **价格询问**
   - 直播间用语："多少米"、"啥价"、"几万"

### 分析结果

- **用户名**: 识别出的有购车意向的用户
- **合并内容**: 将同一用户的多条相关评论合并成一条意向描述
- **精准过滤**: 自动过滤无关评论，只显示汽车相关内容

## 常见问题

### Q: 提取不到评论怎么办？
A: 请确保：
- 当前页面有评论区
- 评论已完全加载（可以滚动到评论区等待加载）
- 使用的是小红书或抖音平台

### Q: AI 分析提示错误？
A: 请检查：
- API Key 是否正确配置
- 网络连接是否正常
- DeepSeek 账户是否有剩余额度

### Q: 如何查看更多评论？
A: 在网页上向下滚动加载更多评论后，重新点击"提取评论"即可

### Q: API Key 安全吗？
A: API Key 仅存储在您的本地浏览器中，不会上传到任何服务器，请放心使用

## 最佳实践

1. **批量分析**: 先多滚动几次页面加载足够多的评论，再进行提取和分析
2. **定期更新**: 如果网站更新导致无法提取，请关注项目更新
3. **API 额度**: 合理使用 AI 分析功能，避免频繁调用浪费额度
